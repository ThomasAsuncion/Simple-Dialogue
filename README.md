# Simple Dialogue

A simple dialogue system made with Unity (ver. 2019.3.8f1). This dialogue system contains the following features:

* Dialogue Trigger
* Named Conversations
* Multiple Lines of Conversations
* Basic animations
* Typerwriter effect on dialogue text
* Skipping prevention
